Utvilking ord og utrykk

HTML
-HTML står for Hyper Text Markup Language.
HTML er standard markup-språk for å lage nettsider.
HTML beskriver strukturen til en webside.
HTML består av en rekke elementer.
HTML-elementer forteller nettleseren hvordan innholdet skal vises.
CSS
- CSS står for Cascading Style Sheets
CSS beskriver hvordan HTML-elementer skal vises på skjerm, papir eller i andre medier
CSS sparer mye arbeid. Den kan kontrollere utformingen av flere nettsider samtidig
Eksterne stilark lagres i CSS-filer
